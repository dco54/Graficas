# Graficas
1) para crear la linea:
  precionar Clic-izquierdo y arrastrar,soltar al finalizar
2) para crear el poligono:
  marcar con Clic-izquierdo cada arista
  Clic-derecho para cerrar el poligono
3) usar el teclado:
  traslacion:
    W=>arriba
    S=>abajo
    A=>izquierda
    D=>derecha
  rotacion:
    {=>sentido antihorario
    }=>sentido horario
  escala:
    +=>aumentar
    -=>reducir
